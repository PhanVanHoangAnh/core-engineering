# Core Engineering Lab

Repository cá nhân dành cho việc rèn luyện **nền tảng khoa học máy tính và kỹ thuật lập trình cốt lõi**.

Mục tiêu của repository này là duy trì và phát triển các kỹ năng lập trình nền tảng độc lập với framework, công cụ tự động hóa hay xu hướng công nghệ ngắn hạn.

Trọng tâm của repo:

- Thuật toán
- Cấu trúc dữ liệu
- Tư duy giải quyết vấn đề
- System-level understanding
- Kiến thức khoa học máy tính cơ bản

Ngôn ngữ chính sử dụng: **C++**

---

# Triết lý của repository

Trong bối cảnh hiện nay, việc phát triển phần mềm ngày càng phụ thuộc nhiều vào:

- Framework
- Thư viện
- Công cụ tự động
- AI

Mặc dù các công cụ này giúp tăng năng suất, nhưng chúng cũng làm giảm cơ hội tiếp xúc với các kiến thức nền tảng của khoa học máy tính.

Repository này được xây dựng như một **phòng luyện tập kỹ thuật cá nhân**, nhằm:

- duy trì tư duy thuật toán
- hiểu sâu cách hệ thống hoạt động
- rèn luyện kỹ năng lập trình thuần
- củng cố kiến thức nền tảng của computer science

# Phương pháp luyện tập

Mỗi buổi học:

1. Đọc lý thuyết
2. Tự implement thuật toán
3. Viết test case
4. Phân tích độ phức tạp
5. So sánh các cách tiếp cận

Thời gian luyện tập đề xuất:

30–60 phút mỗi ngày.
